# PhishGuard

**PhishShield** is a high-fidelity, ultra-modern Cyber-Security Awareness Dashboard designed to train users on identifying and preventing phishing attacks. It features a powerful local heuristic scanner, interactive training modules, and real-time threat intelligence visualization.

- **Phish-Scanner Engine**: A local heuristic analyzer that scans email headers and content for urgency tactics, suspicious links, and brand impersonation.
- **MFA Simulator**: An interactive mini-game to teach users how to handle unexpected 2FA requests.
- **Deep-Link Visualizer**: A "Think Before You Click" tool that reveals hidden malicious URLs on hover.
- **Url X-Ray**: A forensic terminal for deep URL structure analysis.
- **Comparison Lab**: Real vs. Fake email comparison with interactive threat marking.
- **Threat Map**: Visualized holographic globe showing mock global phishing trends.
- **Live Breach Feed**: A scrolling ticker of recent global cybersecurity incidents.
